4. Опис на проектот:
Смарт часовник што континуирано ќе ги мери виталите на корисникот (носачот). Часовникот во првите неколку дена по неговото ставање, ќе учи кои се нормални граници за крвниот притисок, срцевиот ритам и др. на корисникот, така што доколку има значително отстапување од тие граници во одреден момент, прво ќе прати предупредување, а доколку истото не се исклучи рачно од корисникот, ќе повика автоматски брза помош.
